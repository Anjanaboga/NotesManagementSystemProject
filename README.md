# Notes Management System (Flask)

A secure Notes Management Web Application built using Flask, SQLite/MySQL, and Bootstrap.

## Features
- User Registration & Login Authentication
- Password Reset via Email
- Add / Update / Delete Notes
- Search Notes Functionality
- User-specific secure CRUD operations
- Responsive Bootstrap UI

## Tech Stack
- Python (Flask)
- SQLite / MySQL
- HTML, CSS, Bootstrap
- Jinja2 Templates

## Installation
```bash
git clone https://github.com/Anjanaboga/NotesManagementSystemProject.git
cd NotesManagementSystemProject
pip install -r requirements.txt
python app.py
